# ShiftManager

Developer Guide:
At the time of development:

Node.js v 24.11.1 /n
React.js v 5.1.0 /n
PostgreSQL/PGAdmin v 18 /n

To install the latest node version: /n

```
npm install npm@latest -g
```

Make sure to clone the repo from GitHub: /n
git clone https://github.com/Bzenker29/ShiftManager.git

Create your own branch so you do not push directly to main: /n
git branch "branch name" /n
git checkout "branch name" /n

Once you make a push from your branch make sure to get oversight and merge with main: /n
git checkout main /n
git merge branch "branch name" /n
git push origin main /n
